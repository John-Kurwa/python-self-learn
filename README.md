# Python Demo Examples

This repository contains basic Python programming examples covering fundamental concepts.

## Table of Contents

- [Data Types](#data-types)
- [Variables](#variables)
- [Operators](#operators) 
- [Strings](#strings)

## Data Types

Located in [`data_types/`](data_types/) directory:

- Basic built-in data types (int, float, complex, etc.)
- String operations and manipulation
- Type casting examples

## Variables 

Located in [`variables/`](variables/) directory:

- Variable declaration and assignment
- Multiple variable assignment
- Variable unpacking
- Global variables
- Type checking

## Operators

Located in [`operators/`](operators/) directory:

- Arithmetic operators (+, -, *, /, etc.)
- Assignment operators (=, +=, -=, etc.)
- Comparison operators (==, !=, >, <, etc.)
- Logical operators (and, or, not)
- Identity operators (is, is not)

## Strings

Located in [`strings/`](strings/) directory:

- String methods and manipulation
- String formatting
- String operations like:
  - Case conversion
  - Whitespace removal
  - String splitting and joining
  - String searching
  - String alignment

## Getting Started

To run any example:

```python
python filename.py
```

For instance:
```python
python variables/example_1.py
```

## Requirements

- Python 3.x
